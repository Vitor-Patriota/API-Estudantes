# API Estudantes

Projeto simples com backend em Node.js + Express e frontend em React + Vite para cadastro de estudantes.

## Estrutura

- `Projeto-API-Estudantes-BackEnd-main (1)/Projeto-API-Estudantes-BackEnd-main`: API
- `Vitor-Patriota-Projeto-API-Estudantes-FrontEnd-main (2)/Vitor-Patriota-Projeto-API-Estudantes-FrontEnd-main`: interface web

## Como rodar

### Backend

```bash
cd "Projeto-API-Estudantes-BackEnd-main (1)/Projeto-API-Estudantes-BackEnd-main"
npm install
node server.js
```

O backend roda em `http://localhost:3000`.

### Frontend

```bash
cd "Vitor-Patriota-Projeto-API-Estudantes-FrontEnd-main (2)/Vitor-Patriota-Projeto-API-Estudantes-FrontEnd-main"
npm install
npm run dev
```

O frontend roda em `http://localhost:5173`.

## Observação

Para o backend funcionar corretamente, é necessário configurar a conexão com o MongoDB no arquivo `.env`.
